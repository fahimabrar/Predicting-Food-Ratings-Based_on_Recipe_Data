Investigating relation type of most Correlated variables

Abrar

To see the relation types of most correlated values. We selected correlation > 0.85 variables of our EDA

data <- read.csv("FinalSample.csv")
library(ggplot2)
ggplot(data = data, aes(x = FatContent, y = SaturatedFatContent)) + geom_point()

ggplot(data = data, aes(x = FatContent, y = SaturatedFatContent)) + geom_smooth()
## `geom_smooth()` using method = 'gam' and formula 'y ~ s(x, bs = "cs")'

ggplot(data = data, aes(x = FatContent, y = SaturatedFatContent)) + geom_smooth(method = "lm")
## `geom_smooth()` using formula 'y ~ x'

ggplot(data = data, aes(x = FatContent, y = SaturatedFatContent)) + geom_smooth(method = "lm", formula = y ~ poly(x, 3))

ggplot(data = data, aes(x = FatContent, y = Calories)) + geom_point()

ggplot(data = data, aes(x = FatContent, y = Calories)) + geom_smooth()
## `geom_smooth()` using method = 'gam' and formula 'y ~ s(x, bs = "cs")'

ggplot(data = data, aes(x = FatContent, y = SaturatedFatContent)) + geom_smooth(method = "lm")
## `geom_smooth()` using formula 'y ~ x'
